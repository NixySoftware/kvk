# Rust API client for Kamer van Koophandel

Rust OpenAPI client for [Kamer van Koophandel](https://www.kvk.nl/).

For more information, visit https://developers.kvk.nl/.

## Installation

```shell
cargo add kvk
```

## Documentation

Documentation for is available on [Docs.rs](https://docs.rs/kvk/latest/kvk/).
